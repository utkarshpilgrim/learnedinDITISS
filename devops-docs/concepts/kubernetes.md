# Kubernetes Commands

- **`minikube start --driver=qemu --network socker_vmnet`**
- **`minikube docker-env`**
- **`minikube profile list`** - Get the list of current clusters.
- **`minikube logs list`**
- **`minikube service list`**
- **`minikube service <service-name>`**, for example, `minikube service argocd-server -n argocd`, `minikube service mywebsite`.
