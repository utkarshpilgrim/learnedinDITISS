# Public Key Cryptography Standards (PKCS)

It is group of public-key cryptography standards devised and published by RSA security LLC.  The company published the standards to promote the use of the cryptography techniques for which they had patents.

Most of methods of encryption we studied comes into PKCS, for example CSR, Deffie-hellman key agreement standards, private-key information syntax standard, elliptic-curve algorithm, all of them comes under the Public key cryptography Standards. 

# FIPS 140-2

- Fedoral information proessing standard publication 140-2, (FIPS PUB 140-2).

- US govt. computer security standard used to approve cryptography modules, meaing any org that is starting a product needs to implement cryptographuc moules specified by the govt. 

- Specifes the security requirements that will be satisfied by a cryptography module.
