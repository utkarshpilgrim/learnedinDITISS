# split tunnel and full tunnel

| Feature            | Split Tunnel                                 | Full Tunnel                                  |
|--------------------|---------------------------------------------|---------------------------------------------|
| **Traffic Routing** | Some traffic goes through the VPN; some does not. | All traffic goes through the VPN.           |
| **Security Level**  | Less secure for non-VPN traffic; only sensitive data is protected. | More secure as all traffic is encrypted.    |
| **Performance**     | Typically faster for non-sensitive activities since they bypass the VPN. | Can be slower due to all traffic being encrypted and routed through the VPN server. |
| **Use Cases**       | Accessing local resources while connected to a VPN, maintaining speed for general browsing. | Ensuring complete security for all data, such as when using public Wi-Fi. |
